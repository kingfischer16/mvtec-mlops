# mvtec-mlops

MLOps pipeline for industrial anomaly detection using Databricks Community Edition.

## Overview

This project implements a full MLOps pipeline with:
- Medallion architecture (Bronze/Silver/Gold) for data processing
- Image processing for MVTec AD dataset
- ML experimentation with MLflow tracking
- Model deployment

## Tech Stack

- **Platform**: Databricks Community Edition
- **Data**: Delta Lake, PySpark
- **ML**: MLflow, PyTorch/TensorFlow
- **Dataset**: MVTec Anomaly Detection
